What is this

Docket is a small pipeline that reads a receipt and returns its details as clean data. The store, the date, each item, and the total.

It never trusts the reader blindly. Every result is checked against a strict format and scored against a known correct answer. Anything wrong is caught and held back, not passed on.

This page runs the real code on your machine. It uses saved sample results, so it needs no internet and no account.
